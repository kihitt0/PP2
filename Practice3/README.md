## Concepts Covered

### Functions
- **basic_functions.py** — defining functions, parameters, return values, nested calls
- **function_arguments.py** — positional, keyword, and default arguments
- **return_values.py** — returning single/multiple values, lists, dicts, early return, None
- **args_kwargs.py** — `*args`, `**kwargs`, unpacking lists and dicts

### Lambda
- **lambda_basics.py** — lambda syntax, conditionals, IIFE, closures
- **lambda_with_map.py** — `map()` for transforming lists
- **lambda_with_filter.py** — `filter()` for selecting elements, combining filter + map
- **lambda_with_sorted.py** — `sorted()` with custom keys, multi-criteria sorting, `min()`/`max()`

### Classes
- **class_definition.py** — creating classes, methods, attributes
- **init_method.py** — `__init__` constructor, default values, object state
- **class_methods.py** — instance methods, `@classmethod`, `@staticmethod`, method chaining
- **class_variables.py** — class vs instance variables, shared state, counters

### Inheritance
- **inheritance_basics.py** — parent/child classes, `isinstance()`, `issubclass()`
- **super_function.py** — `super()` in constructors and methods, multi-level inheritance
- **method_overriding.py** — polymorphism, `__str__`, `NotImplementedError`
- **multiple_inheritance.py** — multiple parents, MRO (Method Resolution Order), mixins

## How to Run

Each file is a standalone script. Run any file with:

```bash
python Practice3/functions/basic_functions.py
python Practice3/lambda/lambda_basics.py
python Practice3/classes/class_definition.py
python Practice3/inheritance/inheritance_basics.py
```
